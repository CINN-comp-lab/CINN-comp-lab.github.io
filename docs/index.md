# CINN Computational Lab

The **Computational Imaging and Neuroscience (CINN) Lab** at the University of Reading
develops open-source tools for neuroimaging research on the RACC2 HPC cluster.

## Getting Started

New to the cluster? Start here:

- [Connecting to racc2 via SSH and VSCode](tutorials/connecting-to-racc2.md) — set up keyless access from your Mac

## Packages

| Package | Description | Guide |
|---------|-------------|-------|
| [CINN_fmriprep](https://github.com/CINN-comp-lab/CINN_fmriprep) | Run fMRIprep and MRIQC on racc2 via SLURM | [fMRIprep](tutorials/running-fmriprep.md) · [MRIQC](tutorials/running-mriqc.md) |
| [CINN_diskmon](https://github.com/CINN-comp-lab/CINN_diskmon) | Interactive dashboard for monitoring HPC disk usage over time | [Guide](tutorials/diskmon.md) |

## About

These tools are developed and maintained by the CINN Computational Lab at the University of Reading.
